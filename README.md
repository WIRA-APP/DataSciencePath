# WIRA Data Science

Repositori yang mendokumentasikan seluruh workflow Data Science pada proyek WIRA, mulai dari pengumpulan dan integrasi data spasial, analisis eksploratif, feature engineering, hingga pembentukan dataset siap AI untuk mendukung pengambilan keputusan lokasi bisnis berbasis data.

Fokus utama repositori ini adalah mengubah data geografis dan aktivitas bisnis menjadi insight yang dapat digunakan dalam pengembangan sistem rekomendasi lokasi usaha yang lebih objektif dan terukur.

---

## 🎯 Tujuan Proyek

WIRA dikembangkan untuk membantu proses analisis dan pemilihan lokasi bisnis melalui pemanfaatan data spasial, karakteristik jaringan jalan, aksesibilitas transportasi, fasilitas sekitar, serta kondisi persaingan usaha.

Melalui pendekatan Spatial Data Science, data geografis yang tersebar dari berbagai sumber diolah menjadi representasi yang lebih informatif sehingga dapat digunakan sebagai dasar analisis dan pengambilan keputusan bisnis.

---

## 🔄 Workflow Data Science

Repositori ini mencakup seluruh tahapan Data Science yang dilakukan dalam proyek WIRA:

1. Data Acquisition
2. Data Integration
3. Data Cleaning & Validation
4. Data Enrichment
5. Spatial Feature Engineering
6. Business Feature Engineering
7. Exploratory Data Analysis (EDA)
8. Extended Data Analysis (ExDA)
9. Dataset Preparation
10. AI-Ready Dataset Generation
11. Dashboard Development
12. Documentation & Reporting

## 📂 Repository Contents

### Dataset
Berisi dataset hasil integrasi dan pengolahan pada berbagai tahapan pipeline, termasuk dataset hasil preprocessing dan dataset siap pemodelan.

### Notebooks
Kumpulan notebook yang mendokumentasikan proses ekstraksi data, transformasi, feature engineering, EDA, ExDA, dan persiapan dataset.

### Dashboard
Dashboard analitik untuk eksplorasi dan visualisasi insight bisnis yang dihasilkan dari data.

### Reports
Dokumentasi proyek, laporan komprehensif, serta catatan analisis yang mendukung pengembangan sistem WIRA.
