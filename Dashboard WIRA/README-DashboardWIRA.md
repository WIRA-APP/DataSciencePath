## WIRA Dashboard Analytics

Dashboard interaktif WIRA digunakan untuk mengeksplorasi hasil analisis spasial, melihat distribusi potensi bisnis, serta memvisualisasikan berbagai insight yang diperoleh dari proses Data Science dan Artificial Intelligence.

### Menjalankan Dashboard Secara Lokal

Install seluruh dependency yang dibutuhkan:

```bash
pip install -r requirements.txt
```

Jalankan aplikasi Streamlit:

```bash
streamlit run app.py
```

Dashboard akan tersedia pada:

```text
http://localhost:8501
```

### Akses Dashboard Online

Dashboard juga dapat diakses melalui deployment Streamlit Cloud:

https://wira-dashboard.streamlit.app/

### Fitur Dashboard

- Business Opportunity Analysis
- Location Intelligence Visualization
- Business Ecosystem Analytics
- Competitor Distribution Analysis
- Accessibility & Traffic Analysis
- AI-Based Business Recommendation Results
- Interactive Maps & Spatial Insights
